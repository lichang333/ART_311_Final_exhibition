Welcome to my How-To guide for USDA Food Composition Databases API.This guide will help you get started using the USDA Food Composition Databases API. I will do some simple illustrate to tell you how to use that.

USDA Food Composition Databases API is a database which collected many of food and other nutrients' data to help people lookup. It offers Food reports and Nutrient Reports, you can use this search some food information though your phone or web application by using this API. The information of the food or nutrient can be search though NDB number for food (ndbno) and nutrient number(nutrient_no). For example, if you want to make a phone app to calculate how much nutrient that the user got. Use can use this API to get the nutrients information and add together and give a health report to the user according to the user's input information.
