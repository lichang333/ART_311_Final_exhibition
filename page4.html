More Examples

Now, we learned how to use the list API, which are one of the USDA food composition database APIs. But you might ask, how can I use the 3 rest of APIs(Food Reports, Nutrient Reports, Search)?

So, in this stage, I will an example for each of these perspectively.

Example 1: Food Search

This example is to show you how to use the food search API. The user will input some key word text that they want to search and on the serverside will give them back all the results with a JSON file.

Example 3: Nutirient Report

This API is similar to the Food Report API, but this API is more narrow and only get the information for a food's specific nutrient. For example, if I want to know how many Vitamin D3 (the nutrient ID for Vitamin D3 is 326) in a baby food ndbno"03084", I don't need to use Food report API to get the whole nutrient content information. That is very redundant and useless. In constract, The only two things that care about the the product basic information and how much Vitamin D3 in it. So, this API will be really good for that situation.

Response JSON file in key word "326"(nbno) and "03084"(ndbno) :

        {
    "report": {
        "sr": "28",                //Standard Release version of the data being reported
        "groups": "All groups",    //If applicable, the list of food groups used to filter the report
        "subset": "All foods",     //The name of the subset of foods in the response or all foods
        "end": 1,                  //The number of the last item in the reports -- normally start + max.
        "start": 0,                //Value of the offset parameter used in the report
        "total": 1,                //The total number of items available in the reports. Useful for paging the report.
        "foods": [                 //food name
            {
                "ndbno": "03084",
                "name": "Babyfood, dinner, vegetables and turkey, strained",
                "weight": 256.0,
                "measure": "1.0 cup",
                "nutrients": [
                    {
                        "nutrient_id": "326",
                        "nutrient": "Vitamin D3 (cholecalciferol)",
                        "unit": "\u00b5g",
                        "value": "0.0",
                        "gm": 0.0   //The 100 gram equivalent value for the nutrient
                    }
                ]
            }
        ]
    }
}
